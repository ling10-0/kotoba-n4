# Kotoba 日課 — 日文 N4 每日學習 MVP

一個手機優先、完全靜態的日文 N4 每日學習與進度追蹤網站。資料儲存在使用者自己的瀏覽器，不需要帳號或後端。

## MVP 功能

- 昨日完成情況回報
- 每日 5 題混合複習測驗
- 正確率低於 70% 時降為 7 個新詞
- 一般情況維持 10 個新詞
- 連續三次高於 90% 時提升為 12 個新詞
- 每第 7 次學習改為 20 題週複習，不增加新詞
- 每日 3 句造句紀錄
- 易錯詞累積與優先複習
- 學習天數、單字量與平均正確率
- 瀏覽器本機儲存

## 本機執行

```bash
pnpm install
pnpm dev
```

## GitHub Pages

專案已包含 `.github/workflows/deploy-pages.yml`。將 `site` 資料夾內容推送到 GitHub 儲存庫的 `main` 分支，並在儲存庫的 Settings → Pages 將 Source 設為 GitHub Actions；之後每次推送都會自動更新網站。
